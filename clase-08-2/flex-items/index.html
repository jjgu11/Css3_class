Los flex-items de forma predeterminada intentan rellenar todo el espacio disponible en el main-axis. Si el tamaño sumado de todos los flex-items supera al main-size, los items se encogerán (por más que tengan una dimensión declarada, width o height).

"flex box"

Los items son cajas flexibles (flexible boxes), esto significa que pueden crecer o encongerse según las circunstancias. Ignorando sus dimensiones declaradas.

Existen tres propiedades que definen el main-size de un flex-item:

  • flex-basis (indica el tamaño base)
  • flex-grow (define el crecimiento)
  • flex-shrink (define el encogimiento)

Dimensiones de los flex-items

Cross-size

  • Si se ha definido, este se respeta.
  • Si no se ha definido hay dos opciones
    • Si align-items es stretch (o no se define) el cross-size será todo el espacio disponible.
    • Si align-items es diferente de stretch (o los items están varias filas) el cross-size es relativo al contenido y al box-model

Main size

Se calcula con flex-basis, flex-grow y flex-shrink

Flex-basis

  • flex-basis es el tamaño base para el main-size
  • Esto significa que no siempre será el tamaño calculado final
  • flex-basis siempre gana sobre la dimension relativa declarada (excepto si flex-basis es auto)

Flex-grow

  • Indica cuantas unidades crecerá un item
  • Su valor es un número entero positivo
  • El valor de cada unidad se calcula diviendo el espacio disponible entre la suma de unidades totales
  • Por defecto, los flex-items comienzan con flex-grow:0

* El espacio disponible es el espacio total del contenedor menos el espacio ocupado por los items y sus márgenes

Flex-shrink

  • Indica cuantas unidades se encogerá un item
  • Su valor es un número entero positivo
  • El valor de cada unidad se calcula diviendo el espacio disponible entre la suma de unidades totales
  • Por defecto, los flex-items comienzan con flex-shrink:1

Shorthand "flex"

Este shorthand combina flex-grow, flex-shrink y flex-basis (en ese orden).

Valores especiales de flex

  • flex: 0 1 auto (default)
  • flex: auto (1 1 auto)
  • flex: none (0 0 auto)
  • flex: N (N N auto)
